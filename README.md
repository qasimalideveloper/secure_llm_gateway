# secure_llm_gateway
A security gateway for LLM applications built with Flask and Microsoft Presidio
#Follow the following instructions to install it in your pc
1. Download the files
2. Run this command to install the requirements
   pip install -r requirements.txt
3. Run this command to run the server
   python main.py
4. To actually check the api, go inside the api_checker folder and run this command to run the api checker
   python api_checker.py
